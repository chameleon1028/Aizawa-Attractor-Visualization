📐 About the Aizawa Attractor

The Aizawa attractor is a classic example of a strange attractor arising from a three-dimensional autonomous system of ordinary differential equations. It exhibits chaotic behaviour for the parameter set shown below.

ẋ = (z − b)x − d y ẏ = d x + (z − b)y ż = c + a z − z³/3 − (x² + y²)(1 + e z) + f z x³

Default parameters: a=0.95 b=0.7 c=0.6 d=3.5 e=0.25 f=0.1 · Integration: RK4, dt=0.01, 20 000 steps, transient removed. 10 independent initial conditions near the origin all converge onto the same strange attractor.
